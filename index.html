Privacy Policy – Artilingo Academy Dictionary by Utku Lehrer

(Version 2.0 – 2025)
Last updated: July 12, 2025

1. What is this document?

This privacy policy explains how Artilingo Academy Dictionary ("the Application") collects, uses, stores, and protects your data. It is intended for users of the desktop software developed and distributed by Utku Lehrer .

2. What kind of data do we collect?

The application does not collect any personal information . However, for licensing and logging purposes, it processes:

Data Type Description
License Code Encrypted (SHA-256 hashed) form of the code you enter
Role Defines your access level: Admin, Standard Student, or Pro Student
Timestamp Date and time of activation, login, or update-related events
Hash Code Used to anonymously track license activations or update actions

3. Why is this data used?

Purpose Explanation
License verification Ensure that entered codes are valid, not expired, and uniquely assigned
Security Detect multiple usage or license abuse across systems
Log generation Maintain local activation logs on your device (not uploaded elsewhere)
Update system Validate versions before downloading updates from Google Drive

4. Does the app require internet access?

Only for these two purposes:

  • License activation (only once, during setup)
  • Optional update check from a public Google Drive link

📌 No continuous connection, telemetry, or sync with any server is performed.

5. Where is the data stored?

All data is stored locally on your computer in the following structure:

File/Folder Purpose
activation_log.json Stores hash of activation codes and timestamps
data/ Stores your dictionary content (themes, words)
media/ Stores images and sound files for vocabulary

🔐 No data is uploaded, synchronized, or backed up externally.

6. Does this app access Google services?

✅ Yes , but only to download publicly accessible update packages via Google Drive.
It does not use Google Sign-In, OAuth login, Google Docs, Gmail, or access your private Google Drive files.

The included credentials.json file only authorizes download from a specific shared Drive folder using predefined API keys.
No personal Google account data is accessed or required.

7. Who can access the data?

Role Access Scope
Admin Can view activation logs and add/edit/delete content
Student Can only view content; has no access to logs or license info

8. What this app does not collect:

❌ This app does not collect, store, or transmit:

  • Name or surname
  • Email address
  • IP address
  • Location or device information
  • Webcam, microphone, or biometric data
  • Any sensitive or personally identifiable information (PII)

9. Does it use cookies or web trackers?

❌ No. This is a desktop-based offline application . It does not use:

  • Cookies
  • Session IDs
  • Web analytics
  • Tracking pixels
  • Advertisements

10. Can I delete my data?

Yes. You have full control.
Simply delete or reset the following files:

  • activation_log.json (for license history)
  • data/ (for themes and vocabulary)
  • media/ (for pictures and audio)

There is no backup anywhere else. Once deleted, your data is gone.

11. Is this app GDPR-compliant?

✅ Yes.
Because the app does not collect or process personally identifiable data, and all logs are stored locally in hashed (anonymous) format, it complies with GDPR and similar global privacy standards.
